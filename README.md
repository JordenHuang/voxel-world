# Voxel World

Minecraft-like voxel game implement with TypeScript and WebGL, for learning purpose.

You can try it here: [https://jordenhuang.github.io/voxel-world](https://jordenhuang.github.io/voxel-world)

## How to run it

1. Clone the repo
2. run `npm install` to install dependencies
3. run `npm run dev` to start vite server, and go to the url that serves the files
